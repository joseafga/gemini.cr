# gemini

Google Gemini API written in Crystal. It is not complete, it was created for use in another project, but it may be 
useful.

## Installation

1. Add the dependency to your `shard.yml`:

   ```yaml
   dependencies:
     gemini:
       github: joseafga/gemini.cr
   ```

2. Run `shards install`
